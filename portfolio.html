Fruit Catcher

In Fruit Catcher, fruit fall from the sky, and your goal is to catch all the fruits except for the lemons.
We used def functions and a while loop to define the collisions and how to move around the basket.

The Range (Aim Trainer)

The Range is a clicker based game where you have to click on the bot and wingman in order to get positive points. Clicking on your allies and leer gives you negative points.

We used line indexing, manipulating strings, and def functions to help with our leaderboard and point systems.

Halloween Candy Catcher

Halloween Candy Catcher is a game themed off of halloween. The goal of the game is to try and collect as much candy possible using your left and right arrow keys.

Fish Tank Program

The Fish Tank Program is a program we had to debug and change the code, as well as bypass encrypted code to have the program run properly without running into errors. This helped us learn about protecting our cybersecurity.
